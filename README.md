# 💱 Currency Converter

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. It allows users to convert currencies using real-time exchange rates from an API.

## 🚀 Features

- Real-time currency conversion
- Clean and responsive design
- Easy-to-use interface
- Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Currency Exchange API
